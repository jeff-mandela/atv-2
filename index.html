Tudo sobre html

HTML/ HyperText Markup Language é a linguagem usada para criar páginas na internet. Ele funciona como a “estrutura” de um site,dizendo ao navegador quais partes são títulos, parágrafos, imagens, links e outros elementos. Um arquivo HTML é formado por tags, que são palavras especiais usadas para marcar e organizar o conteúdo. Por exemplo, a tag h1 cria um título grande, enquanto a tag p cria um parágrafo de texto.

Sem HTMLos navegadores não saberiam como exibir textos, imagens e links de forma organizada. Por isso, ele é a base de qualquer página web.

Lístas

Estrutura básica de uma página HTML

  1. !DOCTYPE html → avisa que o arquivo é HTML.
  2. html → envolve todo o conteúdo da página.
  3. head → parte “invisível” com informações da página.
  4. title → nome que aparece na aba do navegador.
  5. body → parte “visível” com textos, imagens, links etc.

Elementos básicos para colocar conteúdo

  • h1 → título principal.
  • p → parágrafo de texto.
  • a → link para outra página ou site.
  • img → imagem.
  • ul e li → lista de itens (por exemplo, lista de compras).
